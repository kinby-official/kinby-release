# **Kinby**

## 项目介绍

Kinby 是一款全平台的媒体播放器，致力于提供流畅、高效的用户体验。支持连接 Emby 和 Jellyfin 等流媒体平台，用户可以轻松访问和管理其多媒体内容。Kinby 的界面简洁现代，支持多种设备，包括移动端、桌面端及电视端，适合家庭娱乐、个人影院等场景。无论您身处何地，Kinby 都能带来高速、流畅的观看体验。
